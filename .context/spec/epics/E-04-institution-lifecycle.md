# E-04: Institution Lifecycle

**Feature:** F-02
**Estimated Sprints:** 1
**Sprint Assignment:** Sprint 3

## Definition of Done
- Waitlist application submitted via public form
- SuperAdmin reviews pending applications in queue
- Approve creates institution record + dispatches invitation email to admin
- Reject records reason and notifies applicant
- Institution status lifecycle: pending → approved / rejected
