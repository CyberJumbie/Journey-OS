# E-22: Critic Agent & Review Router

**Feature:** F-10
**Estimated Sprints:** 1
**Sprint Assignment:** Sprint 13

## Definition of Done
- Critic Agent (Claude Opus) scores on 6 quality metrics
- Review Router: auto-approve / auto-reject / route to faculty based on scores
- Three automation levels: Full Auto, Checkpoints, Manual
- Automation level configurable per institution

## User Flows Enabled
- UF-16: Question Review Queue — enhanced with automated routing
- UF-17: Self-Review Own Question — partially enabled (scoring visible)

## Story Preview
- Story: Critic Agent service — Claude Opus 6-metric scoring
- Story: Review Router — threshold-based routing logic
- Story: Automation level configuration — institution-level setting
