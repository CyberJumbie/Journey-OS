# E-19: Workbench UI

**Feature:** F-09
**Estimated Sprints:** 1
**Sprint Assignment:** Sprint 7

## Definition of Done
- Split-screen workbench: 45% chat panel / 55% context panel
- Chat panel with CopilotKit integration for conversational generation
- Context panel shows: source material, concept graph, framework tags
- Mode switcher: generate / bulk / review
- Generation spec wizard for configuring question parameters

## User Flows Enabled
- UF-14: Single Question Generation — enhanced with full workbench UI
- UF-15: Bulk Question Generation — partially enabled (UI only, no bulk backend)

- Story: SplitPane layout — resizable 45/55 split with mode switcher
- Story: ChatPanel component — CopilotKit chat with SSE streaming
