# E-02: Registration & Onboarding

**Feature:** F-01
**Estimated Sprints:** 1
**Sprint Assignment:** Sprint 3

## Definition of Done
- 4-step registration wizard captures role, profile, institution, FERPA consent
- Institutional Admin invitation flow accepts token and creates account
- Persona-specific onboarding screens guide first-time users
- Email verification required before full access

## User Flows Enabled
- UF-01: Faculty Registration & Onboarding — fully enabled
- UF-02: Admin Invitation Onboarding — fully enabled

## Story Preview
- Story: Registration wizard — multi-step form with role selection, profile, FERPA consent
- Story: Invitation acceptance flow — token validation, account creation
- Story: Persona onboarding screens — role-specific welcome + setup guidance
- Story: Email verification gate — verify email before granting full access
