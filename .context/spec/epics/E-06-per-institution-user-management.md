# E-06: Per-Institution User Management

**Feature:** F-03
**Estimated Sprints:** 1
**Sprint Assignment:** Sprint 3

## Definition of Done
- Institutional Admin can list, invite, and deactivate users within their institution
- Faculty invitation sends email with role pre-assigned
- Course Director flag can be toggled per faculty member
- Role assignment enforces valid roles per institution tier

- Story: Role assignment & CD flag — toggle roles, set Course Director permission
